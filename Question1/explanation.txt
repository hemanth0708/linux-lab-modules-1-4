2. Workspace Validation
pwd shows the present working directory.
ls -l lists files and directories in long format with permissions, owner, size, and time.

3. Environment Confirmation File
The echo command writes the confirmation message into user_info.txt.

4. File Integrity Check
wc -c counts the number of characters present in the file.

5. Learning the Tools
The man mkdir command opens the manual page for mkdir.
Useful option: -p
It creates parent directories automatically if they do not exist.

6. Home Directory Inspection
ls ~ lists the contents of the home directory in alphabetical order.

7. Log Investigation
grep searches for the word "admin" in log.txt and displays matching lines only.

8. System Information Check
uname -r displays the currently running Linux kernel version.

9. Network Connectivity Test
ping sends ICMP packets to check network connectivity with www.google.com.

10. System Health Awareness
uptime shows how long the system has been running, number of logged-in users,
and the system load average for the last 1, 5, and 15 minutes.
